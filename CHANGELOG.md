# Changelog

Alle noemenswaardige wijzigingen aan dit project worden in dit bestand
gedocumenteerd. Het formaat volgt losjes [Keep a Changelog](https://keepachangelog.com),
en het project gebruikt [Semantic Versioning](https://semver.org).

## [1.0.0] — Eerste publieke release

### Toegevoegd
- Volledig lokale besturing van de Tjilla C150 via het Tuya 3.3-protocol.
- Kamerreiniging via het lokale `0x14`-commando, geverifieerd op hardware.
- Losse "reinig deze kamer"-knoppen en selectie-switches per kamer.
- Toestand-bewuste besturingsknoppen (start/pauze/stop/dock).
- Optimistische UI-updates met bevestiging via lokale statuspushes.
- Sensoren voor batterij, reiniging, totalen en onderdeel-levensduur.
- Instellingen voor zuigkracht, dweilen, volume, niet-storen en meer.
- Nederlandse en Engelse vertalingen.
- Diagnostics-export met geredacteerde sleutels.

### Opmerking
Eerste stabiele release. Zie de README voor meer informatie.
